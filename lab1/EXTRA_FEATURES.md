# Дополнительные возможности лабораторной работы

Этот документ описывает дополнительные функции, реализованные для получения дополнительных баллов.

## 1. Решение двойственной задачи

### Описание
Программа автоматически строит и решает двойственную задачу для любой исходной задачи линейного программирования, проверяя теорему двойственности.

### Использование
```bash
python dual_solver.py <путь_к_файлу_задачи>
```

### Пример
```bash
python dual_solver.py test_problems/dual_example.txt
```

### Что делает программа
1. Решает исходную (примальную) задачу симплекс-методом
2. Строит двойственную задачу по правилам двойственности
3. Решает двойственную задачу
4. Сравнивает значения целевых функций и проверяет теорему двойственности

### Теоретическая основа
Для задачи:
- **Примальная:** min c^T x при Ax ≥ b, x ≥ 0
- **Двойственная:** max b^T y при A^T y ≤ c, y ≥ 0

Теорема двойственности утверждает, что если обе задачи имеют оптимальные решения, то их значения целевых функций совпадают: Z* = W*.

## 2. Решение задач целочисленного линейного программирования

### Описание
Реализован метод ветвей и границ (Branch and Bound) для решения задач, где переменные должны принимать целочисленные значения.

### Использование
```bash
# Все переменные целочисленные
python integer_solver.py <путь_к_файлу_задачи>

# Только указанные переменные целочисленные (номера 1-based)
python integer_solver.py <путь_к_файлу_задачи> <var1> <var2> ...
```

### Примеры
```bash
# Все переменные целочисленные
python integer_solver.py test_problems/integer_example.txt

# Только x1 и x2 целочисленные
python integer_solver.py test_problems/integer_example.txt 1 2
```

### Алгоритм
1. **Релаксация:** Решаем задачу без требования целочисленности
2. **Проверка:** Если решение целочисленное - готово
3. **Ветвление:** Выбираем переменную с наибольшей дробной частью
4. **Подзадачи:** Создаем две подзадачи:
   - x ≤ ⌊x*⌋
   - x ≥ ⌈x*⌉
5. **Рекурсия:** Рекурсивно решаем подзадачи
6. **Отсечение:** Отсекаем подзадачи, которые не могут дать лучшее решение

### Особенности
- Поддерживает задачи максимизации и минимизации
- Можно указать, какие переменные должны быть целочисленными
- Автоматически отсекает неперспективные ветви (bound)
- Выводит статистику по количеству исследованных узлов

## Тестовые примеры

### Двойственная задача
Файл: `test_problems/dual_example.txt`
```
minimize
4 1 1 2
2 1 0 1 <= 9
1 1 1 0 = 7
0 0 1 1 >= 5
```

### Целочисленное программирование
Файл: `test_problems/integer_example.txt`
```
maximize
3 2
2 1 <= 6
1 2 <= 8
1 0 >= 0
0 1 >= 0
```

Ожидаемое решение: x1 = 2, x2 = 2, Z = 10

## Требования
Все дополнительные модули используют существующую реализацию симплекс-метода из `simplex_solver.py`, поэтому никаких дополнительных зависимостей не требуется.

## Файлы
- `dual_solver.py` - модуль для решения двойственной задачи
- `integer_solver.py` - модуль для решения задач ЦЛП методом ветвей и границ
