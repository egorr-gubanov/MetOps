# История изменений - Дополнительные возможности

## Версия 2.0 - Дополнительные функции для получения баллов

### Добавлено

#### 1. Решение двойственной задачи (`dual_solver.py`)
- Автоматическое построение двойственной задачи из исходной
- Решение двойственной задачи симплекс-методом
- Проверка теоремы двойственности (Z* = W*)
- Поддержка всех типов ограничений (≤, =, ≥)
- Обработка неограниченных переменных

**Использование:**
```bash
python dual_solver.py test_problems/dual_example.txt
```

#### 2. Решение задач целочисленного линейного программирования (`integer_solver.py`)
- Реализация метода ветвей и границ (Branch and Bound)
- Поддержка задач максимизации и минимизации
- Возможность указать, какие переменные должны быть целочисленными
- Автоматическое отсечение неперспективных ветвей
- Статистика по количеству исследованных узлов

**Использование:**
```bash
# Все переменные целочисленные
python integer_solver.py test_problems/integer_example.txt

# Только указанные переменные
python integer_solver.py test_problems/integer_example.txt 1 2
```

### Тестовые примеры
- `test_problems/dual_example.txt` - пример для двойственной задачи
- `test_problems/integer_example.txt` - пример задачи ЦЛП (maximize)
- `test_problems/integer_example2.txt` - пример задачи ЦЛП (minimize)
